# 產品型錄檢索計劃

> 目標：將廠商 PDF 型錄（鋁線槽系列 P2–P15）轉換為可查詢的 Excel 規格表 + 圖庫，
> 供業務報價、LINE 客服回覆、網站產品頁三個情境使用。

---

## 已完成

### 1. Excel 規格表（22 欄全欄位）

**檔案**：`鋁線槽系列規格表.xlsx`
**產生腳本**：`build_excel_aluminum.py`（`python3 build_excel_aluminum.py` 重產）

| 欄位群 | 欄位 |
|--------|------|
| 識別 | 序號、型錄頁、大類、料號、品名 |
| 尺寸 | 外寬(mm)、外高(mm)、電路槽寬(mm)、板厚(mm) |
| 安裝 | 安裝方式、安裝位置 |
| 出光 | 出光方向 |
| 電氣 | 電壓、色溫選項、IP防護等級 |
| 場景 | 適用場景（逗號標籤字串） |
| 特色 | 特殊功能（逗號標籤字串）、可選顏色 |
| 配件 | 對應配件料號 |
| 報價 | 定價(NT$)、計價單位 |
| 備註 | 備註 |

- 共 **92 筆**產品／配件，涵蓋 P002–P015 全部 30 個大類
- 首列凍結 + AutoFilter；純型材的電氣欄填 N/A
- V/S（明裝/嵌入）雙價格以 `"150 / 160"` 字串記入，單位欄標示「明裝/嵌入 /M」

---

### 2. 圖庫骨架（前三款）

**資料夾**：`圖庫/`
**擷取腳本**：`extract_images.py`（`python3 extract_images.py` 重跑）

| 資料夾 | 狀態 | 說明 |
|--------|------|------|
| `圖庫/HB128/` | 已提取，待分類命名 | 19 檔：全頁參考.png + 18 張內嵌圖 |
| `圖庫/HB128X/` | 已提取，待分類命名 | 同 P002 頁面，圖片與 HB128 相同，需手動挑選 |
| `圖庫/HB381A/` | 已提取，待分類命名 | 32 檔：含 P003 整頁（混有其他 4 款產品圖） |

**標準圖片命名規則**（整理後改為以下名稱）：

```
外觀.jpg      ← 產品本體照
截面.jpg      ← 尺寸截面圖
安裝.jpg      ← 施工步驟示意
效果_01.jpg   ← 空間情境照（可多張）
效果_02.jpg
全頁參考.png  ← 保留，作為整理參考用
```

### 圖片來源規劃
| 來源 | 對應圖種 | 做法 |
|------|----------|------|
| PDF 截圖（已有腳本） | 截面、安裝、外觀 | 從 `全頁參考.png` 手動裁切，或寫腳本裁切區域 |
| 廠商 QR 碼影片 | 效果圖 | 掃描 PDF 上 QR 碼，截圖存入 `效果_xx.jpg` |
| 廠商另提供素材 | 全種類 | 按料號放入對應資料夾，改為標準命名 |
| 自行拍攝 | 外觀、效果圖 | 直接存入資料夾，改名即可 |
